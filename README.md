# 📚 Bookish - Library Management Web Application

**Bookish** is a web-based Library Management System built using **ASP.NET Core MVC**.  
The application allows users to manage books in a digital library through a simple and organized interface.

This project demonstrates the use of the **MVC architecture**, **Entity Framework Core**, and **SQL Server** to build a full-stack web application.

---

# 🚀 Features

- 📖 View all books in the library
- ➕ Add new books
- ✏️ Edit book information
- ❌ Delete books
- 🔍 Browse and manage book records
- 🗂 Clean MVC project structure
